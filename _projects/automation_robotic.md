# Autonomous UAV Simulation Project
This project simulates a quadrotor drone performing object detection and autonomous navigation in a 3D world. It combines ArduPilot, ROS 2, Gazebo, YOLO, and Blender assets to create a complete autonomous flight system.

You can find the source code on my GitHub page:  
[https://github.com/EhsanZibaei/ardupilot_yolo_integration](https://github.com/EhsanZibaei/ardupilot_yolo_integration)
